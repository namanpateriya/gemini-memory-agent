# Gemini Memory Agent

Semantic memory agent powered by Google Gemini.

Supports:

- Conversational semantic memory
- Retrieval-augmented responses
- CLI-based interaction
- FastAPI-based APIs

---

## Features

- Google Gemini integration
- Semantic memory retrieval
- Vector memory using FAISS
- Lightweight memory agent architecture
- CLI and API support
- Memory inspection support
- Evaluation support

---

## Why This Repository

Most AI chatbots today are stateless.

This project demonstrates:

- semantic memory
- contextual recall
- retrieval-augmented conversations
- lightweight agentic workflows

without heavy frameworks.
